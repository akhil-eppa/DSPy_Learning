# Learning the basics of DSPy through examples

This repository contains basic, initial use cases of DSPy to better understand the fundamentals
* notebooks/dspy_entity_extraction.ipynb: To help understand concepts of signatures, predictor and optimizers. An entity extraction pipeline is implemented in DSPy and different components of DSPy are explore
* notebooks/dspy_query_assistant.ipynb: A slightly more complex use case involving combining multiple predictors into a module and development of an agent in DSPy and walk through of the same
